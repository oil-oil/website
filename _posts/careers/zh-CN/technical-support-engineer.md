### 岗位职责

1. 为 Apache APISIX、API7 等 API 网关产品，提供售后技术支持工作，解决用户问题，保证系统稳定运行。参与线上值班，确保复杂系统能够稳定地运行；
2. 排查用户环境的问题和故障，进行重现和定位。完善文档和手册，帮助用户更好的使用产品。

### 任职要求

1. 熟悉 Linux 系统的操作；
2. 熟悉 kubernetes 的原理和运维；
3. 熟悉常用 CI/CD 工具；
4. 具备问题定位和重现的能力；
5. 结果导向，良好的沟通能力。
6. 熟悉 API 网关产品，对开源有兴趣的优先。
